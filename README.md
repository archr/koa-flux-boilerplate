# koa-flux-boilerplate

Boilerplate/workflow for an application with react and flux in the client and koa for the server.

## Install

```sh
$ git clone --depth 1 https://github.com/archr/koa-flux-boilerplate appname
```

## Development

For the client:

```sh
$ cd client
$ npm install
$ bower install
$ gulp dev
```

For the server:
```sh
$ cd server
$ gulp
```
